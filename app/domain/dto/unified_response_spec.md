# Спецификация: DTO UnifiedResponse (unified_response.py)

> **Файл реализации:** `unified_response.py`  
> **Слой:** Domain (чистое ядро, 0 внешних зависимостей кроме Pydantic)  
> **Ответственность:** Стандартизированное представление ответа от LLM-провайдера

---

## 1. Общие правила

- DTO — это **неизменяемая** (frozen) Pydantic-модель для передачи данных между слоями.
- Адаптеры обязаны трансформировать вендор-специфичные форматы в этот DTO и обратно.
- Ядро системы работает **исключительно** с этим DTO, никогда — с форматами вендоров.
- Модель должна быть сконфигурирована как неизменяемая (frozen) через механизм конфигурации Pydantic V2.

---

## 2. Вложенная модель: UsageInfo

| Поле               | Тип   | Обязательное | Описание                    |
|--------------------|-------|--------------|-----------------------------|
| `prompt_tokens`    | целое | Да           | Количество токенов промпта  |
| `completion_tokens`| целое | Да           | Количество токенов ответа   |
| `total_tokens`     | целое | Да           | Общее количество токенов    |

### Валидация UsageInfo

- Все поля: неотрицательные целые числа.

---

## 3. DTO: UnifiedResponse

### Назначение

Стандартизированное представление ответа от LLM-провайдера, которое адаптер возвращает ядру.

### Поля

| Поле                | Тип                    | Обязательное | По умолчанию | Описание                                          |
|---------------------|------------------------|--------------|--------------|---------------------------------------------------|
| `trace_id`          | строка                 | Да           | —            | UUID v4 (тот же, что в запросе)                   |
| `content`           | строка                 | Да           | —            | Текст ответа модели                               |
| `model`             | строка                 | Да           | —            | Фактическая модель, которая ответила              |
| `usage`             | UsageInfo или None     | Нет          | None         | Статистика использования токенов                  |
| `provider_raw`      | словарь                | Нет          | пустой словарь| Сырой ответ провайдера (для отладки)             |
| `guardrail_blocked` | булево                 | Нет          | False        | Был ли запрос заблокирован Guardrail              |
| `guardrail_details` | словарь или None       | Нет          | None         | Детали блокировки (если была)                     |

### Валидация

- `trace_id`: формат UUID v4.
- `content`: может быть пустой строкой (если заблокирован Guardrail).

---

## 4. Обработка ошибок

При невалидных данных выбрасывается стандартный pydantic.ValidationError.
Трансформация ошибок в HTTP-ответы — ответственность слоя api/.
